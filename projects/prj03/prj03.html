Catanalysis: an R package

Disclosure: This project is still in its development phase. At the moment, I am building an easy to use R package that will be openly available on the Comprehensive R Archive Network in the coming month(s). This package will take data output from Catscan (detailed below) and allow the user to run a large amount of analysis functions on the data, and create easy to read summary graphics and reports.

Catanalysis is an analysis package for the classification tool CatScan. The reference for this tool can be seen below.
Klippel, A., M. Worboys, and M. Duckham. “Identifying factors of geographic event conceptualisation.” International Journal of Geographical Information Science 22.2 (2008): 183–204.

Screenshot of the CatScan interface of an ongoing mock-up experiment.

CatScan is a classification "card-sorting" tool. Icons of any kind, chosen by the researcher, are placed on the left side of the screen, and empty categories on the right side of the screen. Participants click and drag icons from the left, to categories on the right.
